Pocket_Option_Signal_Bot-
A self-hosted signal generator for Pocket Option OTC assets. The application connects to Pocket Option's live WebSocket feed, runs a multi-indicator technical analysis engine on incoming candle data, and streams directional signals (UP, DOWN, or WAIT) with an associated confidence score to a real-time web dashboard.
